# QuickNews

A React-based news application that fetches real-time news using GNews API.

## Features

- Real-time news updates
- Category-wise news filtering
- Infinite scrolling
- Responsive design
- Bootstrap UI

## Tech Stack

- React.js
- Bootstrap
- GNews API
